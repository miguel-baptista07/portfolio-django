Sobre este Portfolio

Arquitetura MVT

Este portfolio foi desenvolvido em Django, seguindo o padrão MVT (Model-View-Template). O Model define a estrutura dos dados e interage com a base de dados SQLite. A View contém a lógica de negócio — recebe pedidos HTTP, consulta os modelos necessários e devolve uma resposta. O Template define a apresentação HTML, recebendo os dados da View e renderizando-os para o utilizador.
